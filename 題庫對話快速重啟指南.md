## 🐛 常見問題與解決方案

### 問題 1：章節列表顯示空白
**原因**：題庫數據格式不一致
- 一般業/技術士：外部 JSON 陣列 → 需轉換為物件 `{"1": {...}}`
- 甲業/有機溶劑：內嵌在 index.html

**解決**：檢查 `selectSource()` 和 `selectMode()` 函數的數據載入邏輯

### 問題 2：Google OAuth 被刪除
**原因**：本地開發版本覆蓋部署版本時丟失 OAuth 代碼

**解決**：
```bash
# 從 git 歷史恢復
git log --oneline | head -20
git checkout <commit-hash> -- index.html
```

### 問題 3：學習進度顯示錯誤
**原因**：`CHAPTERS` 變數在切換題庫時被覆蓋

**解決**：使用獨立變數保存各題庫數據
- `window.LUO_DATA` - 一般業
- `window.JIA_DATA` - 甲業
- `window.ZHIAN_DATA` - 技術士
- `window.ORGANIC_DATA` - 有機溶劑

### 問題 4：瀏覽器緩存問題
**解決**：強制刷新 `Ctrl+Shift+R` 或在 URL 後加 `?v=時間戳`

## 📝 關鍵代碼邏輯

### 題庫載入流程
```javascript
// selectSource() - 選擇題庫時載入數據
function selectSource(src) {
  CURRENT_SOURCE = src;
  if (src === 'luo') {
    loadChaptersData().then(data => {
      window.LUO_DATA = data;
      CHAPTERS = data;
      goScreen('mode-select');
    });
  } else if (src === 'zhian') {
    fetch('chapter_zhian_index.json')
      .then(r => r.json())
      .then(data => {
        var obj = {};
        data.forEach(ch => { obj[String(ch.id)] = ch; });
        window.ZHIAN_DATA = obj;
        CHAPTERS = obj;
        goScreen('mode-select');
      });
  }
  // ... 其他題庫
}

// selectMode() - 選擇模式時確認數據已載入
function selectMode(mode) {
  if (CURRENT_SOURCE === 'zhian') {
    if (window.ZHIAN_DATA && Object.keys(window.ZHIAN_DATA).length > 0) {
      proceedMode();
    } else {
      // 重新載入...
    }
  }
}
```

### 章節渲染邏輯
```javascript
// selectMode() → proceedMode() → 渲染章節列表
if (src === 'zhian') {
  var keys = Object.keys(window.ZHIAN_DATA).sort((a,b) => parseInt(a)-parseInt(b));
  keys.forEach(k => {
    var ch = window.ZHIAN_DATA[k];
    html += '<div class="chapter-card" onclick="selectChapter(\''+k+'\')">' +
      '<div class="ch-card-num">第 '+k+' 章</div>' +
      '<div class="ch-card-title">'+escHtml(ch.chapter)+'</div>' +
      '<div class="ch-card-q">'+ch.total_questions+' 題</div>' +
      '</div>';
  });
}
```
